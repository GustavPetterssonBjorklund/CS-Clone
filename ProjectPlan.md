# CS-Clone Overview
A Counter-Strike style online PvP shooter.

# Core Gameplay Plan
Bomb defusal focused on 1v1 or small teams. Target 5 rounds to win, with smooth and responsive movement.

# Platform and Networking
Windows PC first, using dedicated authoritative servers.

# Initial Content Scope
Start with one map and an MVP arsenal: AK-47 and grenades. Expand only if time allows and priorities permit.

# Art Style Direction
Low poly art style.
